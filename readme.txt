Данное приложение позволяет делать следующее:
1) Выполнять CRUD-операции над сущностью Person - добавлять, редактировать и удалять людей
2) Добавлять людям детей и родителей и удалять уже имеющихся
3) Просматривать всех детей и всех родителей человека
4) Осуществлять поиск людей по заданным критериям:
   а) По имени, фамилии и интервалу, в котором должен находится день рождения(можно искать только
      по нескольким данным полям, можно по всем)
   б) По ID пасспорта и гражданству - данные два поля вместе имеют уникальные значения в БД,
      поэтому таким образом можно найти только максимум одного человека

Интерфейс PeopleDAO в пакете ru.kpfu.db.ildar.dao определяет операции к базе данных, класс PeopleDAOImpl является реализацией данного интерфейса и предоставляет доступ к БД Oracle через фреимворк Spring JDBC Templates - надстройка JDBC от Spring.
Для работы приложения необходимо иметь установленную БД Oracle на компьютере. Данные для соединения следующие:
    driverClassName = oracle.jdbc.driver.OracleDriver
    url=jdbc:oracle:thin:@localhost:1521/orcl
    username=family_tree
    password=ildar
Данная конфигурация определена в spring-beans.xml в папке src/main/resources. Подразумевается, что экземпляр Oracle имеет название orcl. Для успешного подключения по данной конфигурации необходимо сначала выполнить файл oracle_ddl.sql - он создаст пользователя, необходимые таблицы и хранимые процедуры.
